## NOUGIT-REACT-GRAPHQL-TASK

## Steps for running the task on local:

- First get into the root directory and run `yarn install` for client directory to install all client dependencies.
- Run `npm install` in server directory to install all server dependencies.
- Run `yarn start` in client directory for starting the client server in `localhost:3000`.
- Run `npm start` in server directory for starting the server on `localhost:4000`.
- Run `yarn test` in client directory for running the test cases.
